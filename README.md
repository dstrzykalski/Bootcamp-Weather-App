# Bootcamp-Weather-App

A Basic Weather App Made with the OpenWeather API

## Technologies Used

- HTML
- CSS
- JavaScript
- OpenWeather API

## Features

- Search for current weather by city name
- Display temperature, humidity and weather conditions
